# 🔐 React Login Form

A simple and responsive Login Form built using **React.js** and **CSS**. This project demonstrates the fundamentals of React form handling using **useState**, controlled components, and form submission.

## 🚀 Features

* User-friendly login interface
* Built with React Functional Components
* Form handling using `useState`
* Controlled input fields
* Form validation using the `required` attribute
* Prevents page reload with `preventDefault()`
* Clears input fields after form submission
* Responsive and clean UI
* Easy to understand and beginner-friendly code

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6)
* CSS3
* HTML5
